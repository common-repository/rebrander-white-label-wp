== Description ==

## Rebrander for WordPress Features

Rebrander is made for those who want to customize their WordPress site. 
* Easy replace the default logo with your own on the WordPress login screen.
* Custom background color for the login screen.
* Removes WordPress from title and adds your site name.
* Removes dashboard footer, version name and widgets from Dashboard
* Hide the WordPress admin bar logo.
* Hide admin menus & plugins from other administrators.



## Rebrander
The Rebrander Plugin for WordPress is made for professionals who require an extra level of customization.

* Remove WordPress Update Nags
* Remove admin bar on the fronted.
* Change WordPress Email address and sender name.
* Exciting upcoming features and development.
* Premium Support.





== Installation ==
Rebrander plugin Installation Instructions:

Want to add your custom logo straight away?

1. Upload the Rebrander plugin to your /wp-content/plugins/ directory or through the Plugin admin section under "add new".
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. To change the login logo ipage please update your site logo, login logo will be automatically changed
4. That's it!
